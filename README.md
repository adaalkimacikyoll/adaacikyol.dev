# Ada Alkım Açıkyol — Personal Portfolio

A personal portfolio website built from scratch with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean code.

---

## About

Personal portfolio showcasing my projects, experience, and skills as a Software Engineering student at the University of Europe for Applied Sciences in Potsdam. I'm also the co-founder and CEO of **Taply**, an AI-powered fintech startup.

---

## Sections

| Section | Description |
|---|---|
| Hero | Introduction with animated typing effect |
| About | Background, focus areas, and personal info |
| Projects | Selected work including embedded systems tools, AI apps, and full-stack products |
| Experience | Work history and leadership roles |
| Skills | Tech stack and tools |
| Contact | Contact form and social links |

---

## Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure |
| CSS3 | Styling, animations, responsive layout |
| Vanilla JavaScript | Typing effect, scroll reveal, project filtering |
| Google Fonts | Plus Jakarta Sans, Fira Code |
| GitHub Pages | Deployment |

---

## Features

- ✅ Fully responsive — works on mobile, tablet and desktop
- ✅ Smooth scroll reveal animations
- ✅ Animated typing effect in hero section
- ✅ Project filter by category (AI, Full-Stack, Embedded)
- ✅ Experience timeline
- ✅ No frameworks, no build tools — open `index.html` and it works
- ✅ Soft violet dark theme

---

## Run Locally

No installation needed. Just clone and open:

```bash
git clone https://github.com/adaalkimacikyoll/adaalkimacikyoll.github.io.git
cd adaalkimacikyoll.github.io
open index.html
```

Or simply download the ZIP and open `index.html` in your browser.

---

## Deploy

This site is deployed via GitHub Pages. To deploy your own version:

1. Fork this repository
2. Go to **Settings → Pages**
3. Set source to **main** branch
4. Your site will be live at `https://yourusername.github.io`
```
